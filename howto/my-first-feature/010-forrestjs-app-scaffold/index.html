ForrestJS App's Scaffold

Create a new app.js file and use the following scaffold to kick off a new ForrestJS App:

const forrestjs = require('@forrestjs/core');

forrestjs
  .run({
    trace: 'compact',
    settings: {},
    services: [],
    features: [],
  })
  .catch(console.error);

Once ready, you can run it with:

npx nodemon app
